# 图片增强器

此技能可以处理您的图片和截图，使其看起来更好——更清晰、更锐利、更专业。

## 何时使用此技能

- 提升博客文章或文档中的截图质量
- 在社交媒体上分享前增强图片效果
- 为演示文稿或报告准备图片
- 放大低分辨率图片
- 锐化模糊的照片
- 清理压缩过的图片

## 此技能的功能

1. **分析图片质量**：检查分辨率、锐度和压缩伪影
2. **增强分辨率**：智能放大图片
3. **提高锐度**：增强边缘和细节
4. **减少伪影**：清理压缩伪影和噪点
5. **针对使用场景优化**：根据预期用途（网页、印刷、社交媒体）进行调整

## 提示

- 始终保留原始文件作为备份
- 对截图和数字图像效果最佳
- 可以批量处理整个文件夹
- 如有需要可指定输出格式（PNG 用于质量，JPG 用于更小的文件大小）
- 对于社交媒体，请提及平台以获得最佳尺寸

## 常见用例

- **博客文章**：发布前增强截图
- **文档**：使 UI 截图清晰可见
- **社交媒体**：优化图片用于 Twitter、LinkedIn、Instagram
- **演示文稿**：放大图片用于大屏幕
- **印刷材料**：提高物理媒体的分辨率
