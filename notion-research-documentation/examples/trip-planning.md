# 示例：团体旅行研究和规划

**用户请求**："研究和规划我们朋友们的 3 月日本之旅 - 我们是 6 个人，计划 10 天"

## 工作流程

### 1. 搜索现有笔记
```
Notion:notion-search
query: "Japan travel"
```
找到：日本旅行指南（来自朋友）、东京餐厅、京都寺庙指南

### 2. 获取和提取提示
```
Notion:notion-fetch (3x)
```
**来自先前旅行者的关键信息：**
- 最佳时间：3-4 月（樱花季）
- 必看：东京、京都、大阪
- 预算：$200-300/天（中等范围）
- 提前 3 个月预订住宿
- 到达前获取 JR Pass
- 顶级餐厅：Sushi Dai、Ichiran Ramen、筑地场寿司

### 3. 研究和综合
结合先前旅行者的见解与：
- 航班选项和价格
- 住宿类型（酒店/日式旅馆/Airbnb）
- 交通（JR Pass 必需）
- 10 天行程结构
- 预算分解

### 4. 创建综合计划
```
Notion:notion-create-pages
parent: { page_id: "travel-plans-parent-id" }
pages: [{
  properties: {
    title: "日本之旅 2026 - 3月15-25日（10 天）"
  },
  content: "[完整旅行计划，包含行程、预算、提示...]"
}]
```

## 输出示例

```markdown
# 日本之旅 2026 - 3月15-25日（10 天）

## 旅行概述
**日期**：2026年3月15-25日（樱花季 🌸）
**团体**：6 人 | **预算**：$3,000-4,000/人

## 行程摘要

**第 1-3 天：东京**
- 到达，探索涩谷、原宿、新宿
- 参观筑地市场、皇居、浅草寺
- 体验东京夜生活、teamLab Borderless

**第 4-5 天：箱根**
- 从东京一日游
- 温泉、富士山景观、芦之湖游船

**第 6-8 天：京都**
- 从东京乘子弹头列车
- 伏见稻荷、金阁寺、嵯山竹林
- 艺伎区（祇园）、传统茶道

**第 9-10 天：大阪 + 离开**
- 大阪城、道顿堀美食区
- 奈良一日游（鹿公园、东大寺）
- 从大阪离境航班（KIX）

## 预算分解（每人）
- **航班**：$900-1,200（往返）
- **住宿**：$700-1,000（9 晚）
- **JR Pass**：$280（7 日通票）
- **餐饮**：$500-700（$50-70/天）
- **活动**：$300-400
- **当地交通**：$100
**总计**：$2,780-3,680

## 关键预订
- **航班**：提前 3-4 个月预订
- **酒店**：现在预订（樱花季）
  - 东京：新宿区（2 晚）+ 浅草（1 晚）
  - 京都：祇园区（3 晚）
  - 大阪：难波区（2 晚）
- **JR Pass**：旅行前 2-3 周订购
- **餐厅**：提前 1 周预订（Sushi Dai、高端场所）

## 基本提示

- 在寺庙、日式旅馆、一些餐厅脱鞋
- 火车极其准时 - 不要迟到
- 学习基本短语：arigatou、sumimasen、itadakimasu
```
