# 示例：竞争对手分析

## 用户请求

> "研究竞争对手定价模型并创建比较文档"

## 工作流程执行

### 步骤 1：搜索竞争对手信息

```
Notion:notion-search
query: "competitor pricing"
query_type: "internal"
filters: {
  created_date_range: {
    start_date: "2024-01-01"
  }
}
```

**找到的结果：**
- "竞争对手分析 - Stripe" (Product)
- "市场研究 Q3 2024" (Strategy)
- "定价策略讨论" (Product)
- "竞争对手功能矩阵" (Product)

### 步骤 2：获取和分析

```
Notion:notion-fetch for each page
```

**提取的信息：**

**Stripe**：
- 按需付费：2.9% + 每笔交易 $0.30
- 无月费
- 可享受批量折扣

**Square**：
- 按需付费：2.6% + 每笔交易 $0.10
- 月计划：$29/月，费率更低
- 硬件成本单独计算

**PayPal**：
- 标准：3.49% + 每笔交易 $0.49
- 高级：$30/月 + 2.59% + $0.49
- 多币种费用

**Adyen**：
- Interchange++ 定价模型
- 定制企业定价
- 适用设置费和月费

### 步骤 3：创建比较文档

```

  properties: {

```

## 输出文档

```markdown
# 竞争对手定价分析 - 支付处理器

**日期**：2025年10月14日
**研究团队**：产品策略

## 执行摘要

我们分析了四个主要支付处理器竞争对手：Stripe、Square、PayPal 和 Adyen。定价模型从简单的按需付费（Stripe）到复杂的 interchange-plus（Adyen）不等。关键见解：所有竞争对手都为高交易量的商家提供批量折扣，分界点通常为每月 $10 万处理量。

## 比较矩阵

| 功能 | Stripe | Square | PayPal | Adyen |
|---------|--------|--------|--------|-------|
| **基础费率** | 2.9% + $0.30 | 2.6% + $0.10 | 3.49% + $0.49 | Interchange++ |
| **月费** | $0 | $0-29 | $0-30 | 定制 |
| **批量折扣** | 是，>$80K | 是，>$250K | 是，>$100K | 是，定制 |
| **设置费** | $0 | $0 | $0 | $1,000-5,000 |
| **多币种** | 额外 1% | 额外 3% | 额外 3-4% | 包含 |
| **退款费用** | $15 | $15-25 | $20 | 定制 |
| **目标市场** | 初创企业-企业 | 小型-中型 | 小型-中型 | 企业 |

## 详细分析

### Stripe

**定价结构**：
- **标准**：2.9% + 每次成功卡支付 $0.30
- **批量折扣**：适用于处理量 >$80,000/月的企业
- **国际卡**：+1% 费用
- **货币转换**：高于市场汇率 1%

**优势**：
- 简单、透明的定价
- 无设置费或月最低消费
- 优秀的开发者体验
- 快速入驻

**劣势**：
- 高交易量的每笔交易费用较高
- 批量折扣不如 Adyen 激进

**最适合**：需要快速集成的初创企业和成长阶段公司

**来源**：<mention-page url="...">竞争对手分析 - Stripe</mention-page>

### Square

**定价结构**：
- **按需付费**：每次点击、插入或刷卡 2.6% + $0.10
- **手动输入**：3.5% + $0.15
- **Plus 计划**：$29/月，费率更低（2.5% + $0.10）
- **Premium 计划**：定制定价

**优势**：
- 面对面交易的每笔交易费用最低
- 一体化硬件 + 软件
- 无长期合同

**劣势**：
- 在线/手动交易费率较高
- 硬件成本（$49-$299）
- 不太适合仅在线业务

**最适合**：实体零售和餐厅

**来源**：<mention-page url="...">市场研究 Q3 2024</mention-page>

### PayPal

**定价结构**：
- **标准**：每笔交易 3.49% + $0.49
- **高级**：$30/月 + 2.59% + $0.49
- **Payments Pro**：直接信用卡处理额外 $30/月

**优势**：
- 庞大的客户群（PayPal 结账）
- 买家保护增加信任
- 国际覆盖（200+ 国家）

**劣势**：
- 每笔交易费用最高
- 复杂的费用结构
- 账户冻结和储备金常见

**最适合**：PayPal 品牌信任度重要的业务（电子商务、市场）

**来源**：<mention-page url="...">定价策略讨论</mention-page>

### Adyen

**定价结构**：
- **Interchange++**：实际 interchange + 计划费用 + 固定加价
- **设置费**：$1,000-5,000（可协商）
- **月最低消费**：通常 $10,000+ 处理量
- **每笔交易**：Interchange + 0.6% + $0.12（示例）

**优势**：
- 规模下最透明的成本结构
- 高交易量的实际费率最低
- 真正的多币种（100+ 货币）
- 直接连接到计划

**劣势**：
- 定价复杂需要分析
- 高最低消费（$10K+/月）
- 集成时间较长
- 不适合小企业

**最适合**：月处理量 $1M+/月的企业

**来源**：<mention-page url="...">竞争对手功能矩阵</mention-page>

## 定价趋势和见解

### 基于批量的折扣

所有竞争对手都提供规模折扣：
- **入门点**：$80K-$250K/月处理量
- **典型折扣**：10-30 个基点减少
- **谈判杠杆**：$500K/月开始

### 需要考虑的隐藏成本

| 成本类型 | Stripe | Square | PayPal | Adyen |
|-----------|--------|--------|--------|-------|
| 退款 | $15 | $15-25 | $20 | $15-25 |
| 账户验证 | $0 | $0 | $0 | 不同 |
| PCI 合规 | $0 | $0 | $0 | $0 |
| 货币转换 | 1% | 3% | 3-4% | 0% |
| 退款费用 | 返还 | 返还 | 不退还 | 不同 |

### 市场定位

```
高交易量 / 企业
    ↑
    |                    Adyen
    |
    |         Stripe
    |
    |  Square    PayPal
    |
    └──────────────────→
      小型 / 简单        复杂 / 国际
```

## 战略影响

### 对于初创企业（<$100K/月）
**推荐**：Stripe
- 启动阻力最低
- 无前期成本
- 优秀的文档
- 此规模下可接受的费率

### 对于成长中公司（$100K-$1M/月）
**推荐**：Stripe 或 Square
- 谈判批量折扣
- 如果涉及国际，评估 interchange++
- 如果面对面交易占主导，考虑 Square

### 对于企业（>$1M/月）
**推荐**：Adyen 或协商的 Stripe
- Interchange++ 模型节省显著
- 直接计划连接
- 多区域能力重要
- 集成复杂性的投资回报

## 建议

1. **立即**：将我们当前的 2.8% + $0.25 与 Stripe 标准进行基准比较
2. **短期**：在我们当前 $150K/月处理量下向 Stripe 请求批量折扣报价
3. **长期**：当我们跨过 $1M/月门槛时评估 Adyen
```

1. **结构化比较**：矩阵格式便于快速浏览
