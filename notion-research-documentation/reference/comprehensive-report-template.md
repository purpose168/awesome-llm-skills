# 综合报告模板

用于需要广泛分析的深入研究。有关何时使用此格式的信息，请参阅 [comprehensive-report-format.md](comprehensive-report-format.md)。

```markdown
# [报告标题]

## 执行摘要
[一段总结整个报告的文字]

## 背景与上下文
[为什么进行这项研究，它解决了哪些问题]

## 方法论
- 参考来源：[团队空间] 中的 [数量] 个 Notion 页面
- 时间范围：[如适用]
- 范围：[包含/排除的内容]

## 主要发现

### [主要主题 1]
**摘要**：[一句话]

**详细信息**：
- [带有证据的支持点]
- [带有证据的支持点]
- [带有证据的支持点]

**来源**：[页面提及]

### [主要主题 2]
[重复结构]

## 数据与证据

[表格、引用、具体数据点]

## 影响

### 短期
[直接影响]

### 长期
[战略影响]

## 建议

### 优先级 1：[高优先级行动]
- **内容**：[具体行动]
- **原因**：[理由]
- **方式**：[实施方法]

### 优先级 2：[中优先级行动]
[重复结构]

## 附录

### 附加资源
- [相关页面]

### 未解决的问题
- [供未来研究未回答的问题]
```
