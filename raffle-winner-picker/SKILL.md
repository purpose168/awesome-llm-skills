# 抽奖获奖者选择器

此技能从列表、电子表格或Google Sheets中随机选择获奖者，用于赠品和比赛。

## 何时使用此技能

- 运行社交媒体赠品活动
- 在活动中抽取抽奖获奖者
- 随机选择调查或测试的参与者
- 从比赛提交作品中选择获奖者
- 公平分配有限名额或资源
- 随机团队分配

## 此技能的功能

1. **随机选择**：使用加密随机选择
2. **多种来源**：支持CSV、Excel、Google Sheets或纯列表
3. **多个获奖者**：可以选择一个或多个获奖者
4. **防止重复**：确保同一人不能两次获奖
5. **透明结果**：清晰显示选择过程
6. **获奖者详情**：显示关于获奖者的所有相关信息

## 如何使用

### 从Google Sheets

```
从此Google Sheet中随机选择一行以选择赠品获奖者：[Sheet URL]
```

### 从本地文件

```
从entries.csv中随机选择3个获奖者
```

### 从列表

```
从此列表中随机选择一个获奖者：
- Alice (alice@email.com)
- Bob (bob@email.com)
- Carol (carol@email.com)
...
```

### 多个获奖者

```
从contest-entries.xlsx中随机选择5个获奖者，确保无重复
```

## 功能

### 公平选择
- 使用安全随机数生成
- 无偏见或模式
- 透明过程
- 可使用种子重复（用于验证）

### 排除
```
随机选择一个获奖者，排除之前的获奖者：Alice, Bob, Carol
```

### 加权选择
```
根据"entries"列选择加权概率的获奖者（1个条目=1张票）
```

### 候补获奖者
```
从列表中选择1个获奖者和3个候补获奖者
```

## 示例工作流程

### 社交媒体赠品
1. 将Google表单中的条目导出到Sheets
2. "从[Sheet URL]中随机选择一个获奖者"
3. 验证获奖者详情
4. 带时间戳公开发布

### 活动抽奖
1. 创建参与者姓名和邮箱的CSV
2. "从attendees.csv中随机选择10个获奖者"
3. 导出获奖者列表
4. 直接给获奖者发送邮件

### 团队分配
1. 获取参与者列表
2. "将此列表随机分成4个相等的团队"
3. 审查分配
4. 分享团队名单

## 提示

- **记录过程**：保存时间戳和方法
- **公开发布**：分享选择细节以保持透明度
- **检查资格**：验证获奖者符合比赛规则
- **准备备选**：选择候补获奖者以防获奖者不符合资格
- **导出结果**：保存获奖者列表以供记录
