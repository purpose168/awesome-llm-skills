# 任务创建模板

从规范创建任务时使用。

```markdown
# [任务名称]

## 上下文
作为 <mention-page url="...">功能规范</mention-page> 实施的一部分

实施计划: <mention-page url="...">实施计划</mention-page>

## 描述
[需要完成的内容]

## 验收标准
- [ ] [标准 1]
- [ ] [标准 2]

## 技术细节
[技术方法或说明]

## 依赖关系
- 被阻塞于: [任务] 或 无
- 阻塞: [任务] 或 无

## 资源
- [设计链接]
- [相关代码链接]

## 进度
[实施过程中更新]
```
