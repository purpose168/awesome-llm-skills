# 域名头脑风暴器

这个技能通过生成创意选项并检查哪些域名实际上可注册，帮助您为项目找到完美的域名。

## 何时使用此技能

- 启动新项目或新公司
- 推出产品或服务
- 创建个人品牌或作品集网站
- 现有项目重新命名
- 为副业项目注册域名
- 当首选域名被占用时寻找可用替代方案

## 此技能的功能

1. **理解您的项目**：分析您正在构建的内容以及面向的用户群体
2. **生成创意名称**：创建相关且易记的域名选项
3. **检查可用性**：验证哪些域名在多个顶级域名下实际可用
4. **多种扩展名**：推荐.com、.io、.dev、.ai、.app等扩展名
5. **提供替代方案**：如果首选域名被占用，提供变体选项
6. **品牌见解**：解释为什么某些名称效果良好

## 域名命名技巧

### 好域名的特点

✓ **简短**：理想情况下不超过15个字符
✓ **易记**：容易回忆和拼写
✓ **可发音**：可以在对话中轻松说出
✓ **描述性**：暗示您的业务内容
✓ **可品牌化**：足够独特以脱颖而出
✓ **无连字符**：更易于口头分享

### 顶级域名（TLD）指南

- **.com**：通用、可信，非常适合企业
- **.io**：科技创业公司、开发者工具
- **.dev**：面向开发者的产品
- **.ai**：AI/ML产品
- **.app**：移动或Web应用程序
- **.co**：.com的替代方案
- **.xyz**：现代、创意项目
- **.design**：创意/设计机构
- **.tech**：科技公司

## 工作流程示例

### 创业公司启动
1. 描述您的创业想法
2. 获取10-15个跨顶级域名的域名建议
3. 查看可用性和价格
4. 选择前3个最喜欢的
5. 立即注册

### 个人品牌
1. 分享您的姓名和职业
2. 获取变体（firstname.com、firstnamelastname.dev等）
3. 同时检查社交媒体账号可用性
4. 在多个平台注册一致的品牌

### 产品命名
1. 描述产品和目标市场
2. 获取创意、可品牌化的名称
3. 检查商标冲突
4. 验证域名和社交媒体可用性
5. 在目标受众中测试名称

## 成功提示

1. **迅速行动**：好域名很快就会被抢注
2. **注册变体**：获取.com和.io以保护品牌
3. **避免数字**：口头交流时难以传达
4. **检查社交媒体**：确保@username也可用
5. **大声朗读**：测试是否易于发音
6. **检查商标**：确保没有法律冲突
7. **长远考虑**：5年后这个域名是否仍然合理？

## 价格参考

在推荐域名时，我会注明：
- 标准域名：约$10-15/年
- 溢价顶级域名（.io、.ai）：约$30-50/年
- 已被占用域名：如有列出则显示市场价格
- 溢价域名：数百到数千美元
